# 翰墨精英 (Elite Business) - 华为表盘

> 一款面向 HUAWEI WATCH GT 系列的经典商务风格表盘

![表盘预览](docs/preview_full.png)

## 设计理念

**翰墨精英** 以传统商务腕表为灵感，采用罗马数字时标、金色点缀、米白色表底的经典搭配，融合现代智能穿戴的数据展示能力，打造兼具传统美学与实用功能的商务表盘。

### 与常见商务表盘的差异化设计

| 特征 | 常见商务表盘 | 翰墨精英 |
|------|-------------|---------|
| 时标 | 阿拉伯数字 | **罗马数字 (衬线体)** |
| 色调 | 黑白/蓝白 | **米白底 + 金色点缀** |
| 信息面板 | 散落分布 | **双面板对称布局** (公历/农历) |
| 数据行 | 单行或子表盘 | **双行紧凑排列** (步数\|卡路里\|距离 + 心率\|电量) |
| 进度指示 | 弧形填充 | **底部金色弧线** (步数目标进度) |
| 指针风格 | 标准箭头 | **藏青锥形 + 金色中线 + 钢蓝秒针** |

## 功能展示

### 数据显示项

- **天气信息**: 天气图标 + 温度 + 天气文字描述
- **公历日期**: 星期 + 月/日
- **农历日历**: 农历日期 + 时辰
- **数字时间**: HH:MM 格式辅助显示
- **运动数据**: 步数 / 卡路里 / 距离
- **健康数据**: 实时心率
- **设备状态**: 电池电量百分比
- **步数进度**: 底部弧形进度条

## 快速开始

### 环境要求

- Python 3.6+ (带 Pillow 库)
- Bash shell
- zip 命令行工具

### 生成资源

```bash
# 安装依赖
pip install Pillow

# 生成所有图片资源
python3 tools/generate_assets.py
```

### 打包

```bash
# 打包为 .hwt 文件
bash tools/package.sh
```

### 安装到手表

1. 将 `output/Elite_Business.hwt` 传输到手机
2. 打开**华为运动健康** APP
3. 进入 **设备** > **表盘市场** > **我的**
4. 选择已下载的表盘文件，安装到手表
