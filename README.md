# 翰墨精英 (Elite Business) - 华为表盘

> 一款面向 HUAWEI WATCH GT 系列的蓝白商务风格表盘

![表盘预览](docs/preview_full.png)

## 设计理念

**翰墨精英** 以经典蓝白商务腕表为灵感，采用白银渐变表底、蓝色阿拉伯数字时标、蓝色圆点刻度环和深蓝金属质感指针，融合天气、日历（公历+农历）、运动健康等数据显示，打造简洁优雅的商务表盘。

### 设计特色

- **白银渐变底色**：中心纯白，边缘渐变至浅银灰，富有层次感
- **蓝色数字时标**：阿拉伯数字 1-12，无衬线粗体，清晰易读
- **蓝色圆点刻度环**：每分钟一个小蓝点，整点位置加大并配短刻度线
- **双截式指针**：时针/分针前部红色、尾部深色，两截式剑形设计
- **双面板对称布局**：左面板（农历/时辰）+ 右面板（星期/日期），蓝色细边框
- **天气信息区**：天气描述 + 竖线分隔 + 温度/温度范围
- **紧凑数据行**：步数 | 距离 + 心率（红色）

## 功能展示

### 数据显示项

- **天气信息**: 天气描述 + 温度 + 温度范围
- **农历日期**: 农历日期 + 时辰（左面板）
- **公历日期**: 星期 + 月/日（右面板）
- **数字时间**: HH:MM 粗体居中
- **运动数据**: 步数 / 距离
- **健康数据**: 心率

## 快速开始

### 环境要求

- Python 3.6+ (带 Pillow 库)
- Bash shell
- zip 命令行工具

### 生成资源

```bash
# 安装依赖
pip install Pillow

# 生成所有图片资源
python3 tools/generate_assets.py
```

### 打包

```bash
# 打包为 .hwt 文件
bash tools/package.sh
```

### 安装到手表

1. 将 `output/Elite_Business.hwt` 传输到手机
2. 打开**华为运动健康** APP
3. 进入 **设备** > **表盘市场** > **我的**
4. 选择已下载的表盘文件，安装到手表
